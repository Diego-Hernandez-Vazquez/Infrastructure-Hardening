# Infrastructure Hardening & Defensive Security

Bienvenido a mi repositorio central dedicado al endurecimiento de infraestructura (*Hardening*) y la seguridad defensiva. Este espacio documenta las auditorías de seguridad, análisis de superficies de ataque y la implementación de controles técnicos para mitigar vulnerabilidades críticas en entornos empresariales.

El enfoque primordial de estos proyectos es la transición de sistemas vulnerables hacia arquitecturas bajo el principio de **Seguridad desde el Diseño (Security by Design)** y el **Privilegio Mínimo**.

## Núcleo de Especialización Defensiva
* **Endurecimiento de Sistemas:** Remediación de configuraciones por defecto, desactivación de servicios obsoletos y aseguramiento de políticas locales.
* **Seguridad Perimetral:** Configuración avanzada de firewalls a nivel de host, segmentación y filtrado estricto de tráfico entrante/saliente.
* **Auditoría de Cumplimiento:** Identificación activa de brechas de seguridad mediante escaneos detallados y validación post-remediación.
